## 🚀 服务器部署步骤

### 第 1 步：登录服务器

```bash
ssh root@你的服务器 IP
```

### 第 2 步：拉取代码

```bash
cd /root  # 或其他目录
git clone <您的仓库地址>
cd xiang
```

### 第 3 步：配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑配置文件
nano .env
```

**必须修改的配置：**

```bash
# 1. 生成 JWT_SECRET（至少 32 字符）
openssl rand -base64 32
# 将输出值复制到 .env 中的 JWT_SECRET

# 2. 修改数据库密码（可选，建议修改）
DB_PASSWORD=your_secure_password
```

### 第 4 步：一键部署

```bash
# 启动服务（后台运行）
docker-compose up -d --build

# 查看服务状态
docker-compose ps

# 查看日志（可选）
docker-compose logs -f app
```

### 第 5 步：访问网站

在浏览器打开：
```
http://服务器 IP:5700
```

**默认管理员账户：**
- 用户名：`admin`
- 密码：`myhx0308`

## ⚠️ 注意事项

1. **JWT_SECRET** - 必须修改为至少 32 字符的随机字符串
2. **数据库密码** - 建议修改为强密码
3. **首次登录后** - 立即修改管理员密码
4. **防火墙配置** - 仅开放 5700 端口
5. **HTTPS** - 生产环境建议配置 SSL 证书

## 📞 故障排查

如果遇到问题，请按以下步骤排查：

1. **查看容器状态**
   ```bash
   docker-compose ps
   ```

2. **查看应用日志**
   ```bash
   docker-compose logs app
   ```

3. **查看数据库日志**
   ```bash
   docker-compose logs postgres
   ```

4. **检查端口占用**
   ```bash
   netstat -tunlp | grep 5700
   ```

5. **检查环境变量**
   ```bash
   cat .env
   ```
